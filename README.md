# 🛡️ Thief Detection Alarm using AI

An AI-powered **real-time object detection web application** built with **Next.js**, **React**, and **TensorFlow.js (COCO-SSD)**. The application uses your device's webcam to detect people and common objects. When a **person** is detected, it triggers an alarm to simulate a basic theft detection system.

## 🌐 Live Demo

🚀 **Live Website:** https://friendly-dragon-131d48.netlify.app/

## ✨ Features

* 🎥 Live webcam object detection
* 🤖 AI-powered detection using TensorFlow.js COCO-SSD
* 🚨 Alarm sound when a person is detected
* 📦 Detects multiple everyday objects
* ⚡ Fast browser-side inference
* 📱 Responsive UI
* 🔒 No backend or database required
* 🌐 Runs completely in the browser

## 🎯 How It Works

1. User grants webcam access.
2. TensorFlow.js loads the COCO-SSD model.
3. Video frames are analyzed in real time.
4. Detected objects are highlighted with bounding boxes.
5. If a **person** is detected, an alarm sound is played automatically.
